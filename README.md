# Agregator 🌐

Un agrégateur de flux polyvalent et stylisé | A versatile and stylized feed aggregator

---

## À propos | About

**Agregator** est un outil web conçu pour agréger, organiser et afficher des flux de données (RSS, liens, API personnalisées) dans une interface **synthwave** et intuitive. 
| **Agregator** is a web tool designed to aggregate, organize, and display data feeds (RSS, links, custom APIs) in a **synthwave**-styled, intuitive interface.

**Centralisez** vos flux préférés (actualités, réseaux sociaux, données personnelles).
| **Centralize** your favorite feeds (news, social media, personal data).

**Personnalisez** l’affichage avec des thèmes rétro.
| **Customize** the display with retro themes.

**Filtrez** et triez les informations selon vos besoins.
| **Filter** and sort information as needed.

---

## Fonctionnalités | Features

- ☀️ Agrégation de flux RSS | RSS feed aggregation
- 🌍 Interface bilingue (FR/EN) | Bilingual interface (FR/EN) 
- 📍 Thèmes synthwave | Synthwave themes
- 🌐 Interface bilingue (Français/Anglais) | Bilingual interface (French/English)  
- 🕘 Recherche et filtrage | Search and filtering
- 📊Historique des flux | Feed history
- ⏳ Responsive design | Responsive design
- ⚠️ Gestion des erreurs | Error handling
